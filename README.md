
# Paper Rock Scissor game

## Quickstart
npm install -g yarn
yarn install
yarn start

#### Linting
yarn lint

#### Unit tests
yarn test

#### System tests
yarn cucumber

#### run linting and all tests
yarn all

#### depoy on linux using docker-compose
deploy:docker:linux
